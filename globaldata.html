Global Data and Preprocessing

List of datasets used in global dengue case analysis and preprocessing done towards it for visualization.

The global analysis is focused on comparing the severity of the dengue outbreak in different countries and continents based on the average incidence of dengue, and aims to determine the factors that potentially result in the differing levels of severity between the various countries or continents.


Global Dengue Cases Global Population Gross Dometic Product per capita Countries Global Average Precipitation Global Average Temperature

Global dengue cases across the years (Our World in Data).csv

Provides the total number of dengue cases in each country for each year between 1990 to 2019. The dataset was obtained from link and contains following variables:

Fields Description
Entity Country name.
Code Three-letter country code.
Year Year expressed in the format yyyy and ranges from 1990 to 2019.
Incidence - Dengue - Sex: Both - Age: All Ages (Number): Total number of dengue cases in the specified country and year.

This dataset is pivotal for the analysis as the number of dengue cases, after accounting for population sizes (i.e., average incidence, which the team used for more accurate representation), provides a good indication of the severity of dengue outbreak.

API_SP.POP.TOTL_DS2_en_csv_v2_4898931.csv

Describes the population sizes for each country in each of the years between 1960 - 2021. The dataset uses the de facto definition of the total population, which counts all residents regardless of legal status or citizenship and the values shown are mid-year estimates. The dataset was obtained from link and contains the following variables:

Fields Description
Country Name Name of the country./td>
Country Code Three-letter country code.
Indicator Name name of the indicator used; the value for all rows is “Population, total”.
Indicator Code code of the indicator used; the value for all rows is “SP.POP.TOTL”.
Years 1960 - 2021 contains the population size of the country during the respective year.

This dataset would be used to obtain the average incidence of dengue for each country in each year, which the team deems would be a more accurate representation of the severity of dengue outbreaks (refer to Section 6.1.1. for more details).

API_NY.GDP.PCAP.CD_DS2_en_csv_v2_4898793.csv

describes the GDP per capita (current US$) for each country in each of the years between 1960 - 2021. The dataset defines GDP per capita as the gross domestic product divided by the mid-year population. The GDP per capita values are in current US dollars. The dataset was obtained from link and contains the following variables:

Fields Description
Country Name Name of the country.
Country Code Three-letter country code.
Indicator Name Name of the indicator used; the value for all rows is “GDP per capita (current US$)”.
Indicator Code Code of the indicator used; the value for all rows is “NY.GDP.PCAP.CD”.
Years 1960 - 2021 Contains the GDP per capita (current US$) in the country during the respective year.

This dataset would be useful for the analysis of the correlation between GDP per capita (i.e., often used as an indicator of economic status) and the severity of dengue outbreak in the country or continent.

Countries_codes_and_coordinates.csv

provides the average latitude and longitude for each country. The dataset was obtained from link and contains the following variables:

Fields Description
Country Name of the country.
Alpha-2 code Two-letter country code.
Alpha-3 code Three-letter country code.
Numeric code Numeric country code.
Latitude (average) Average latitude of the country.
Longitude (average) Average longitude of the country.

This dataset will be used for the team’s visualisations that include maps, which are useful for illustrating the locations of each country or continent.

API_AG.LND.PRCP.MM_DS2_en_csv_v2_5186842.csv

Describes the average precipitation in depth (mm per year) for each country in each of the years between 1960 - 2021. The dataset was obtained from link and contains the following variables and contains the following variables:

Fields Description
Country Name Name of the country.
Country Code Three-letter country code.
Indicator Name Name of the indicator used; the value for all rows is “Average precipitation in depth (mm per year)”.
Indicator Code Code of the indicator used; the value for all rows is “AG.LND.PRCP.MM”.
Years 1960 - 2021 Contains the average precipitation in depth (mm per year) in the country during the respective year.

This dataset would be useful for the analysis of the correlation between precipitation levels and the severity of dengue outbreak in the country or continent.

GlobalLandTemperaturesByCountry.csv

Provides the average temperature (in degrees Celsius) each month (for years 1743 - 2013) for each country. The dataset was obtained from link and contains the following variables:

Fields Description
dt first day of each month, in the yyyy-mm-dd format.
AverageTemperature Average land temperature during the specified month and year for the country.
AverageTemperatureUncertainty 95% confidence interval around the corresponding AverageTemperature values.
Country Country name.

This dataset would be useful for the analysis of the correlation between temperature and the severity of dengue outbreak in the country or continent.
