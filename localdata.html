Local Data and Preprocessing

List of datasets used in local dengue case analysis and preprocessing done towards it for visualization.

The local analysis is focused on analysing dengue outbreak in Singapore from (year to year) and to compare various factors such as population size of different subzones and planning areas, climate, number and types of dwelling units and water bodies in Singapore. From this analysis, we are able to determine any relationship between the different factors and number of dengue cases in different areas. More attention and specific recommendations can be designed and given for those areas.

Singapore Dengue Cases

This dataset provides the number of dengue cases at various street addresses in Singapore from May 2013 to November 2020. It is obtained from link and contains the following variables:

Fields Description
Number of cases Number of reported dengue cases at the location.
Street Address Street address where dengue cases are reported (down to the apartment block level).
Latitude Latitude of the street address.
Longitude Longitude of the street address
Cluster Number To represent the dengue cluster number
Recent Cases in Cluster Number of dengue cases with onset in last 2 weeks
Total cases in cluster Total number of dengue cases reported in this cluster
Date Date string in YYMMDD format
Month Number Index number of the month, where 1=January and 12=December

Singapore Residents by Subzone and Type of Dwelling, June 2016.shp

This dataset provides the resident population by planning area and subzones based on URA MP14 and type of dwelling in 2016. The shapefile is used to obtain the geometry for the different subzones in Singapore. The shapefile can be obtained from link.

Resident Dwellings by Planning Area of Residence and Subzones 2000 to 2020

This dataset provides the number of HDB dwellings, condominiums, other apartments, landed properties and other types of properties based on planning area of residence and subzones from 2000 to 2020. It is obtained from link and contains the following variables:

Fields Description
Planning Area of Residence
Subzone
Total
HDB Dwellings Total, 1-and 2-Room Flats, 3-Room Flats, 4-Room Flats, 5-Room and Executive Flats
Condominiums and Other Apartments
Landed Properties
Others

Dengue incidence rate 2012 - 2021 excel file

This dataset contains the dengue incidence rate in Singapore from the years 2012 to 2021. The resident populations per year is taken from Residents by Age Group and Type of Dwelling, Annual dataset from link and the yearly number of dengue cases is taken from link. It contains the following variables:

Fields Description
Year
Number of dengue cases
Residents Population
Dengue incidence rate per 100,000 population
